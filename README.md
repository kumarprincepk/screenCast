
Privacy Policy — Screen Cast Chrome Extension

Last Updated: May 2026

Introduction

Screen Cast is a Chrome extension designed to help users record their screen directly from the browser. This Privacy Policy explains how the extension handles user data and permissions.

By using the Screen Cast extension, you agree to the practices described in this policy.

Information Collection

Screen Cast does not collect, store, sell, or share any personal information.

The extension operates entirely within the user’s browser and does not transmit recordings or personal data to external servers.

Permissions Used

The extension requests certain Chrome permissions in order to provide screen recording functionality.

Screen Recording Permission

Used to: capture the selected screen, tab, or window. record screen activity chosen by the user. The extension only records content after the user explicitly grants permission.

Camera Permission

Used to: display webcam preview functionality during recording. The webcam is accessed only when the user starts recording and grants permission.

Microphone Permission

Used to: record microphone audio during screen recording. Microphone access is enabled only after user approval.

Downloads Permission

Used to: save recorded video files directly to the user’s device. Recordings are stored locally on the user’s computer.

Data Storage

Screen Cast does not upload recordings to cloud servers.

All recordings:

remain on the user’s device
are downloaded locally
are controlled entirely by the user

The extension does not maintain external databases or remote storage systems.

Third-Party Services

Screen Cast does not use:

analytics services
advertising networks
tracking systems
third-party data collection tools
Data Sharing

No user data, recordings, or personal information are shared with third parties.

Security

The extension is designed to minimize data exposure by processing recordings locally within the browser environment.

Children’s Privacy

Screen Cast is not directed toward children under the age of 13 and does not knowingly collect personal information from children.

Any updates will be reflected on this page with a revised “Last Updated” date.
